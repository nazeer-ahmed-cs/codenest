Every element rendered by the browser is a rectangular box. The **box model** defines how the size and spacing of each box is calculated. Understanding it is essential for creating predictable layouts.

## The Four Layers

From the inside out, each box is made of four concentric layers:

```
┌─────────────────────────────────┐
│          Margin (outer)         │
│  ┌───────────────────────────┐  │
│  │        Border             │  │
│  │  ┌─────────────────────┐  │  │
│  │  │      Padding        │  │  │
│  │  │  ┌───────────────┐  │  │  │
│  │  │  │   Content     │  │  │  │
│  │  │  │  (text/img)   │  │  │  │
│  │  │  └───────────────┘  │  │  │
│  │  └─────────────────────┘  │  │
│  └───────────────────────────┘  │
└─────────────────────────────────┘
```

1. **Content** — the actual text, image, or child elements
2. **Padding** — transparent space between content and border
3. **Border** — a visible or invisible edge surrounding the padding
4. **Margin** — transparent space outside the border, pushing other elements away

```css
.box {
  width: 200px;
  height: 100px;
  padding: 20px;
  border: 4px solid #2563eb;
  margin: 30px;
}
```

## Visualising Each Property

### Padding

Padding creates breathing room inside the element. It inherits the background color.

```css
.card {
  background: #f9fafb;
  padding: 24px;
  /* Content won't touch the edges */
}
```

Shorthand values follow a clockwise pattern:

```css
padding: 16px;             /* all sides */
padding: 16px 32px;        /* top/bottom  left/right */
padding: 8px 16px 24px;    /* top  left/right  bottom */
padding: 8px 12px 16px 20px; /* top  right  bottom  left */
```

Individual properties: `padding-top`, `padding-right`, `padding-bottom`, `padding-left`.

### Border

Border sits between padding and margin. It requires three values: width, style, and colour.

```css
.box {
  border: 2px solid #2563eb;
}
```

Styles include `solid`, `dashed`, `dotted`, `double`, `groove`, `ridge`, `inset`, `outset`, and `none`.

You can target individual sides:

```css
.card {
  border-bottom: 3px solid #2563eb;
  border-left: 1px solid #e5e7eb;
}

/* Rounded corners */
.card {
  border-radius: 12px;
}
```

### Margin

Margin creates space **outside** the element, pushing neighbouring elements away.

```css
.heading {
  margin-bottom: 24px;
}

.button + .button {
  margin-left: 12px;
}
```

**Margin collapsing:** when two vertical margins meet, the larger one wins — they don't add together.

```html
<div style="margin-bottom: 30px;">Block A</div>
<div style="margin-top: 20px;">Block B</div>
<!-- Gap between them is 30px, not 50px -->
```

## The `box-sizing` Property

By default, `width` and `height` apply only to the **content box**. Padding and border are added on top, making the element larger than expected.

```css
/* Default behaviour (content-box) */
.box {
  width: 200px;
  padding: 20px;
  border: 4px solid black;
  /* Actual width = 200 + 20 + 20 + 4 + 4 = 248px */
}
```

With `box-sizing: border-box`, the width includes padding and border.

```css
.box {
  box-sizing: border-box;
  width: 200px;
  padding: 20px;
  border: 4px solid black;
  /* Actual width = 200px (padding & border are inside) */
}
```

It is standard practice to set this on every element:

```css
*,
*::before,
*::after {
  box-sizing: border-box;
}
```

## Width, Height, and Overflow

Block elements stretch to fill their parent's width by default. You can constrain them:

```css
.container {
  max-width: 960px;
  margin: 0 auto;   /* centres the box horizontally */
}

.card {
  min-height: 200px;
  max-height: 400px;
  overflow-y: auto;  /* scrolls if content overflows */
}
```

## Putting It Together

```html
<div class="card">
  <h2>Box Model in Action</h2>
  <p>Each property below is annotated so you can see how padding, border, and margin affect the layout.</p>
  <button class="btn">Click Me</button>
</div>

<div class="card card--alt">
  <h2>Another Card</h2>
  <p>Notice the margin between these two cards.</p>
  <button class="btn btn--outline">Learn More</button>
</div>
```

```css
* { box-sizing: border-box; }

.card {
  max-width: 400px;
  margin: 24px auto;
  padding: 24px;
  border: 1px solid #e5e7eb;
  border-radius: 12px;
  background: white;
}

.card--alt {
  border-color: #2563eb;
  border-width: 2px;
}

.card h2 {
  margin: 0 0 8px;
  font-size: 1.25rem;
}

.card p {
  margin: 0 0 16px;
  color: #6b7280;
  line-height: 1.5;
}

.btn {
  display: inline-block;
  padding: 10px 20px;
  border: none;
  border-radius: 6px;
  background: #2563eb;
  color: white;
  font-size: 0.875rem;
  cursor: pointer;
}

.btn--outline {
  background: transparent;
  border: 2px solid #2563eb;
  color: #2563eb;
}
```
