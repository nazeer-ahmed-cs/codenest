The `background` property is one of CSS's most versatile tools. It controls what appears behind an element's content and padding areas.

## Background Color

The simplest background is a solid colour.

```css
.notice {
  background-color: #fef3c7;
}
```

Use `background-color` when you only need colour. Reserve the shorthand `background` for combined values.

## Background Image

You can set an image as the background using the `url()` function.

```css
.hero {
  background-image: url("hero-photo.jpg");
}
```

The image tiles by default — it repeats both horizontally and vertically if smaller than the element.

## Background Repeat

Control whether and how the image repeats.

```css
.bg {
  background-image: url("pattern.png");
  background-repeat: no-repeat;      /* appears once */
  background-repeat: repeat-x;       /* horizontal only */
  background-repeat: repeat-y;       /* vertical only */
  background-repeat: repeat;         /* default — both directions */
}
```

## Background Position

Position the image within the element using keywords, percentages, or pixel values.

```css
.bg {
  background-image: url("photo.jpg");
  background-repeat: no-repeat;
  background-position: center;         /* centred both axes */
  background-position: top right;      /* top-right corner */
  background-position: 50% 50%;        /* same as center */
  background-position: 20px 40px;      /* 20px from left, 40px from top */
}
```

Single-value keywords (`center`, `top`, `bottom`, `left`, `right`) centre the other axis automatically.

## Background Size

Control the dimensions of the background image.

```css
.bg {
  background-image: url("photo.jpg");
  background-size: cover;      /* fills the element, may crop */
  background-size: contain;    /* fits entirely, may leave gaps */
  background-size: 100% auto;  /* stretch to full width, auto height */
  background-size: 50px;       /* explicit width, auto height */
}
```

`cover` is the most common choice for hero sections — it ensures the image fills the entire area without distortion.

## Background Attachment

Controls whether the background scrolls with the page or stays fixed.

```css
.parallax {
  background-image: url("mountains.jpg");
  background-attachment: fixed;   /* stays in place while content scrolls */
  /* Use sparingly — can cause performance issues on mobile */
}
```

## The Shorthand Property

All of the above can be combined into a single `background` declaration.

```css
.card {
  background: #1f2937 url("pattern.png") no-repeat center / cover;
}
```

Order: `color image repeat position / size attachment`.

## Multiple Backgrounds

You can layer multiple backgrounds by separating them with commas. The first layer sits on top.

```css
.hero {
  background:
    linear-gradient(rgba(0, 0, 0, 0.5), rgba(0, 0, 0, 0.5)),
    url("hero-bg.jpg") center / cover no-repeat;
}
```

This creates a dark overlay on top of the background image — a common pattern for ensuring text readability.

## Gradients

Gradients are treated as background images. CSS provides two types.

### Linear Gradient

Blends colours along a straight line.

```css
.button {
  background: linear-gradient(135deg, #2563eb, #7c3aed);
}
```

Angle or direction comes first, followed by colour stops.

| Direction | Result |
|---|---|
| `to bottom` | Top to bottom (default) |
| `to right` | Left to right |
| `45deg` | Angled |
| `to top left` | Diagonal |

### Radial Gradient

Blends colours outward from a central point.

```css
.spotlight {
  background: radial-gradient(circle, #fef3c7, #f59e0b);
}
```

### Gradient Colour Stops

You can add multiple colour stops and control where they appear.

```css
.stripes {
  background: linear-gradient(
    90deg,
    #2563eb 0%,
    #7c3aed 50%,
    #f59e0b 100%
  );
}
```

```html
<section class="hero">
  <div class="hero-content">
    <h1>Explore the Mountains</h1>
    <p>Background images, gradients, and overlays working together.</p>
    <a href="#" class="btn">Start Exploring</a>
  </div>
</section>

<section class="features">
  <div class="card card-1">Solid Background</div>
  <div class="card card-2">Gradient Background</div>
  <div class="card card-3">Pattern Overlay</div>
</section>
```

```css
  background:
    linear-gradient(rgba(0, 0, 0, 0.4), rgba(0, 0, 0, 0.6)),

    center / cover no-repeat;
  min-height: 350px;
  display: flex;
  align-items: center;

  text-align: center;

}
```
