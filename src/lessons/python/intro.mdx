Python is a high-level, interpreted programming language known for its readability and versatility. It's used in web development, data science, automation, and more.

## Hello, Python

```python
print("Hello, World!")
```

## Variables

Python is dynamically typed — you don't need to declare a variable's type.

```python
name = "Alice"       # string
age = 25             # int
height = 5.6         # float
is_student = True    # bool

print(f"{name} is {age} years old.")
```

## Comments

```python
# This is a single-line comment

"""
This is a
multi-line comment
(actually a string literal not assigned to anything)
"""
```

## Dynamic Typing

A variable can be reassigned to a different type:

```python
x = 10
x = "now a string"  # perfectly valid
```

## Basic Input

```python
name = input("Enter your name: ")
print(f"Hello, {name}!")
```
