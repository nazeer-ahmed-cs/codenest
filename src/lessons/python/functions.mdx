Functions are reusable blocks of code that take inputs, do work, and return outputs.

## Defining a Function

```python
def greet(name):
    """Return a greeting string."""  # docstring
    return f"Hello, {name}!"

print(greet("Alice"))   # Hello, Alice!
```

## Parameters

```python
# default parameter
def power(base, exp=2):
    return base ** exp

print(power(3))     # 9
print(power(3, 3))  # 27

# keyword arguments
def describe(name, age, city):
    print(f"{name} is {age} from {city}")

describe(age=30, city="NYC", name="Alice")
```

## Variable-Length Arguments

```python
# *args — positional (tuple)
def sum_all(*nums):
    return sum(nums)

print(sum_all(1, 2, 3, 4))   # 10

# **kwargs — keyword (dict)
def print_info(**info):
    for key, value in info.items():
        print(f"{key}: {value}")

print_info(name="Alice", age=30)
```

## Returning Multiple Values

```python
def min_max(nums):
    return min(nums), max(nums)

low, high = min_max([3, 1, 4, 1, 5])
print(low, high)   # 1 5
```

## Lambda Functions

One-line anonymous functions:

```python
square = lambda x: x ** 2
print(square(5))   # 25

# often used with map/filter
nums = [1, 2, 3, 4]
squared = list(map(lambda x: x**2, nums))
evens = list(filter(lambda x: x % 2 == 0, nums))
```

## Scope

Variables defined inside a function are local:

```python
x = 10  # global

def my_func():
    x = 20  # local — shadows global
    print(x)

my_func()   # 20
print(x)    # 10
```
