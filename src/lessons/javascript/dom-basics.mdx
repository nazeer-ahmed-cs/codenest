The **DOM (Document Object Model)** is JavaScript's interface to HTML. It represents the page as a tree of objects you can read and modify.

## What is the DOM?

When the browser loads an HTML page, it creates a structured tree of nodes. Each element, attribute, and piece of text becomes a node you can access with JavaScript.

```html
<html>
  <body>
    <h1>Title</h1>
    <p>Text</p>
  </body>
</html>
```

## Selecting Elements

### `getElementById`

Fastest way to select a single element by its `id`.

```js
let heading = document.getElementById("main-title");
```

### `querySelector`

Selects the first element that matches a CSS selector.

```js
let firstLink = document.querySelector("a");
let button = document.querySelector(".btn-primary");
let sidebar = document.querySelector("#sidebar");
```

### `querySelectorAll`

Returns a static NodeList of all matching elements.

```js
let allLinks = document.querySelectorAll("nav a");
let items = document.querySelectorAll(".list-item");
```

You can loop over a NodeList with `forEach`.

```js
allLinks.forEach((link) => {
  console.log(link.href);
});
```

### Other selection methods

```js
document.getElementsByClassName("card");  // live HTMLCollection
document.getElementsByTagName("p");       // live HTMLCollection
```

Prefer `querySelector` / `querySelectorAll` for consistency — they accept any CSS selector.

## Reading & Changing Content

### `textContent`

Gets or sets the text inside an element (strips HTML tags).

```js
let el = document.querySelector("p");
el.textContent = "Updated text";
console.log(el.textContent);
```

### `innerHTML`

Gets or sets the HTML markup inside an element. Use with caution (XSS risk).

```js
el.innerHTML = "<strong>Bold text</strong>";
```

## Working with Attributes

```js
let img = document.querySelector("img");

img.getAttribute("src");       // "logo.png"
img.setAttribute("alt", "Logo");
img.hasAttribute("src");       // true
img.removeAttribute("title");
```

Direct properties exist for common attributes:

```js
img.src = "new-logo.png";
img.alt = "New Logo";
```

## Changing Styles

Access inline styles via the `style` property. Properties use camelCase instead of kebab-case.

```js
let box = document.querySelector(".box");

box.style.backgroundColor = "blue";
box.style.fontSize = "18px";
box.style.display = "flex";
```

For reading computed styles (including from CSS classes):

```js
getComputedStyle(box).color; // "rgb(0, 0, 0)"
```

## Working with Classes

```js
let el = document.querySelector("div");

el.classList.add("active");
el.classList.remove("hidden");
el.classList.toggle("expanded");
el.classList.contains("active"); // true/false
```

## Creating & Inserting Elements

```js
let newLi = document.createElement("li");
newLi.textContent = "Item 3";
newLi.classList.add("list-item");

let list = document.querySelector("ul");
list.appendChild(newLi);          // at end
list.prepend(newLi);              // at beginning
list.insertBefore(newLi, list.children[1]); // before index 1
```

### Removing elements

```js
newLi.remove();            // modern
list.removeChild(newLi);   // older
```

## Traversing the DOM

```js
let el = document.querySelector("li");

el.parentElement;           // parent node
el.children;                // HTMLCollection of children
el.firstElementChild;       // first child element
el.lastElementChild;        // last child element
el.nextElementSibling;      // next sibling
el.previousElementSibling;  // previous sibling
```
