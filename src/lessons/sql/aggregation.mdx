Aggregate functions compute a single result from a set of rows.

## Common Aggregate Functions

```sql
SELECT COUNT(*) FROM users;           -- total rows
SELECT COUNT(email) FROM users;       -- non-null emails

SELECT SUM(amount) FROM orders;       -- total of all amounts
SELECT AVG(price) FROM products;      -- average price
SELECT MIN(price) FROM products;      -- lowest price
SELECT MAX(price) FROM products;      -- highest price
```

## GROUP BY

Group rows before applying aggregations:

```sql
SELECT category, COUNT(*) AS product_count
FROM products
GROUP BY category;

-- result:
-- Electronics | 12
-- Books       | 8
-- Clothing    | 15
```

## GROUP BY with Multiple Columns

```sql
SELECT category, status, COUNT(*) AS count
FROM orders
GROUP BY category, status;
```

## HAVING

Filter groups after aggregation (WHERE filters rows before grouping):

```sql
SELECT category, AVG(price) AS avg_price
FROM products
GROUP BY category
HAVING AVG(price) > 50;

-- WHERE filters rows, HAVING filters groups
SELECT category, COUNT(*) AS cnt
FROM products
WHERE price > 0                     -- exclude free items
GROUP BY category
HAVING COUNT(*) >= 5;               -- only categories with 5+ products
```

## Complete Example

```sql
SELECT
    customer_id,
    COUNT(*) AS order_count,
    SUM(amount) AS total_spent,
    AVG(amount) AS avg_order_value
FROM orders
WHERE order_date >= '2024-01-01'
GROUP BY customer_id
HAVING COUNT(*) > 3
ORDER BY total_spent DESC;
```
