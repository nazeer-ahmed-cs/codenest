# CodeNest

[![Vercel](https://img.shields.io/badge/deployed%20on-Vercel-000000?logo=vercel)](https://codenest-pink.vercel.app)

An interactive web development learning platform. Learn HTML, CSS, JavaScript, and React through hands-on lessons with live code editors, topic quizzes, and a certification exam.

**Live site:** [codenest-pink.vercel.app](https://codenest-pink.vercel.app)

## Features

- **Interactive lessons** — each lesson includes a live code editor (Sandpack) so you can edit and preview code instantly
- **Structured curriculum** — a curated path from HTML basics through CSS, JavaScript, and React
- **Topic quizzes** — reinforce your knowledge with quizzes at the end of each topic
- **Certification exam** — pass a 30-minute exam with 70%+ to earn a verifiable certificate
- **Code playground** — experiment with HTML, CSS, and JavaScript in a sandboxed environment
- **Reference section** — quick lookup for HTML tags, CSS properties, and JavaScript APIs

## Tech Stack

- **Framework:** [Next.js](https://nextjs.org) (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Live code editor:** Sandpack (CodeSandbox)
- **Authentication:** NextAuth.js
- **Database:** MongoDB
- **Deployment:** Vercel

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.
